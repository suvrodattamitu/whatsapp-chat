== Description ==

Finally we introduce Elementor supported chats.You can set the Live Chat the way you thought.
For Elementor Experts we have Elementor Widget.It needs just two clicks to activate a chat prompt on your website. 
You can customize everything of the chats.If you don't want to use elementor we have this option too.You can activate chat
prompt from settings page.In a word we made it for all kinds of non tech user who can easily integrate chat system on their website
without any trouble.
